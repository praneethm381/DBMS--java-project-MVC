Database Application








Project

About:

This project enables us to perform two operations on database.
1. Fetch all the students who have the department name and total credits same as given in input fields.
2. Fetch the metadata of the tables present in the database.

Usage:

1. Fetch students: This operation needs two fields as inputs: department name and total credits
2. Fetch metadata: Metadata of a table is obtained by selecting a table from the drop down.
    There is also an option ALL to fetch the metadata of all the tables in database.
